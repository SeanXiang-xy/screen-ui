# SLineChart

组件描述

### 基本用法

:::demo 这里可以添加`参数`的描述。

```vue
<template>
  <div class="comp-wrapper">
    <s-line-chart :data="data"></s-line-chart>
  </div>
</template>

<script setup>
import { ref } from "vue";

const data = ref([
  { name: "demo1", value: 123 },
  { name: "demo3", value: 221 },
  { name: "demo2", value: 111 },
]);
</script>

<style>
.comp-wrapper {
  width: 100%;
  height: 300px;
}
</style>
```
