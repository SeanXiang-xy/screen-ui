# SWorkdemo

组件描述

### 基本用法

:::demo 这里可以添加`参数`的描述。

```vue
<template>
  <div class="comp-wrapper">{{ data }}</div>
  <s-workdemo></s-workdemo>
</template>

<script setup>
import { ref } from "vue";

const data = ref("s-workdemo");
</script>

<style>
.comp-wrapper {
  border: 1px solid red;
}
</style>
```
